-_-_-_-_-_-_-_-_-_-_-_- Data Types -_-_-_-_-_-_-_-_-_-_-_-

int - used to store integers of size 4 bytes (depends on the machine type, i.e, 32-bit or 64-bit)
float - used to store numbers with decimal values of size 4 bytes
char - used to store characters of size 1 byte
bool - used to store binary values (true or false) of size 1 byte

Syntax:-

	dataType variableName;	//declaration
	variableName = someValue;	// initialization

	dataType variableName = someValue;	//declaration and initialization



-_-_-_-_-_-_-_-_-_-_-_- Type Modifiers -_-_-_-_-_-_-_-_-_-_-_-

short - used for storing small sized datatypes (mostly int)
unsigned - used to store only non-negative numbers, just the range increases
signed - used to all type of numbers, range is decreased in it
long - used to store large sized datatypes

Syntax:-
	
	typeModifier dataType variableName;
